SWMM5 Lego Builder

3D Toolbar — drop-in CSS for your existing app

Full toolbar
Grid:
All color variants
How to integrate — just add data-color and the class
/* Before (flat): */
<button class="toolbar-btn">Run SWMM5</button>

/* After (3D Lego): */
<button class="lego-btn" data-color="blue">
  <span class="icon">🚀</span> Run SWMM5
</button>

/* Small variant for grid buttons: */
<button class="lego-btn sm" data-color="white">30×30</button>

/* Active/selected state: */
<button class="lego-btn sm active" data-color="yellow">20×20</button>

/* The 3D effect comes from 4 things:
   1. border-bottom: 5px solid <dark shade>    → depth
   2. box-shadow: inset 0 2px 0 <light shade>  → highlight
   3. ::before pseudo-element                    → stud
   4. :active translateY(3px)                    → press
*/
