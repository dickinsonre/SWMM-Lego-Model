The score is limited by the single-file architecture, incomplete LEGO visual theme, and lack of persistence.

## What Works Exceptionally Well

**The dual-engine approach** — offering both a JavaScript animation engine and full EPA SWMM5 via WASM — is exactly right. The JS engine gives instant visual feedback during exploration; the EPA engine gives publication-quality results. The UI clearly distinguishes them with 🚀 (JS) and 🔬 (EPA).

**The 49-storm library** is the most comprehensive free design storm collection available. Having SCS Types I/IA/II/III, regional US standards (FDOT, TxDOT, UDFCD), and international standards (DWA, STOWA, ARR) all pre-calibrated is genuinely valuable.

**The SWMM5 Map legend** (surfaces →`[SUBCATCHMENTS]`, etc.) is brilliant pedagogy. It teaches the SWMM5 file format by showing exactly how each visual element maps to the`.inp` sections.

**Right-click property editing** — discovered via the tutorial, not immediately obvious — allows per-cell override of CN, Manning's n, and other properties. This is essential for realistic modeling and rare in visual editors.

## Improvement 2 — Persistence is Critical

The live app has no save mechanism visible. Every model is lost on refresh. Add a save/load panel:

```jsx
// In the toolbar, between FIX and RUN SWMM5
<div style={{ display: 'flex', gap: 8 }}>
  <button onClick={() => saveToSlot(0)} title="Save to Slot 1">
    💾 Save
  </button>
  <button onClick={() => setShowLoadModal(true)} title="Load Saved Model">
    📂 Load
  </button>
</div>
```

Store up to 5 named slots in localStorage:

```js
const SLOTS_KEY = 'swmm5-lego-slots';

function saveToSlot(slotIndex, name) {
  const slots = JSON.parse(localStorage.getItem(SLOTS_KEY) ?? '[]');
  slots[slotIndex] = {
    name,
    savedAt: new Date().toISOString(),
    grid,
    gridSize,
    stormIdx,
    cellProps,
  };
  localStorage.setItem(SLOTS_KEY, JSON.stringify(slots));
}

function loadFromSlot(slotIndex) {
  const slots = JSON.parse(localStorage.getItem(SLOTS_KEY) ?? '[]');
  if (slots[slotIndex]) {
    setGrid(slots[slotIndex].grid);
    setGridSize(slots[slotIndex].gridSize);
    // ... etc
  }
}
```

Also add an **autosave on every change** with debounce:

```js
useEffect(() => {
  const timer = setTimeout(() => {
    localStorage.setItem('swmm5-lego-autosave', JSON.stringify({
      grid, gridSize, stormIdx, cellProps,
    }));
  }, 1000); // 1 second debounce
  return () => clearTimeout(timer);
}, [grid, gridSize, stormIdx, cellProps]);
```

## Improvement 3 — Results Need Time-Series Export

After running simulation, users want the data. Add CSV export to each results tab:

```jsx
// In the System Results tab header
<div style={{ display: 'flex', justifyContent: 'space-between', marginBottom: 8 }}>
  <h3>System Hydrograph</h3>
  <button
    onClick={() => exportResultsCsv(simResult)}
    style={{ fontSize: 12, padding: '4px 8px' }}
  >
    📥 Export CSV
  </button>
</div>
```

The export function:

```js
function exportResultsCsv(simResult, type = 'system') {
  let rows, filename;

  if (type === 'system') {
    rows = [
      ['Time (min)', 'Rainfall (in/hr)', 'Runoff (CFS)', 'Pipe Flow (CFS)', 'Outfall (CFS)'],
      ...simResult.timeLabels.map((t, i) => [
        t,
        simResult.systemHistory.rainfall[i],
        simResult.systemHistory.runoff[i],
        simResult.systemHistory.pipeFlow[i],
        simResult.systemHistory.outfall[i],
      ]),
    ];
    filename = 'swmm5-lego-system-results.csv';
  } else if (type === 'nodes') {
    rows = [
      ['Node ID', 'Time (min)', 'Depth (ft)', 'Inflow (CFS)', 'HGL (ft)'],
      ...simResult.nodeHistory.flatMap((node, ni) =>
        simResult.timeLabels.map((t, ti) => [
          node.id, t, node.depth[ti], node.inflow[ti], node.head[ti],
        ])
      ),
    ];
    filename = 'swmm5-lego-node-results.csv';
  }

  const csv = rows.map(r => r.join(',')).join('\n');
  const blob = new Blob([csv], { type: 'text/csv' });
  const url = URL.createObjectURL(blob);
  const a = document.createElement('a');
  a.href = url;
  a.download = filename;
  a.click();
  URL.revokeObjectURL(url);
}
```
